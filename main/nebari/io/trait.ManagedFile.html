Expand description

A file that can be interacted with using async operations.

This trait is an abstraction that mimics tokio-uring’s File type, allowing for a non-uring implementation to be provided as well. This is why the read/write APIs take ownership of the buffer – to satisfy the requirements of tokio-uring.

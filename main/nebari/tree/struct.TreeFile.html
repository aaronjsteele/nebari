Expand description

An append-only tree file.

Generics

  • F: An ManagedFile implementor.
  • MAX_ORDER: The maximum number of children a node in the tree can contain. This implementation attempts to grow naturally towards this upper limit. Changing this parameter does not automatically rebalance the tree, but over time the tree will be updated.
